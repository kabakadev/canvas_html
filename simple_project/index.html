Step 2: Make a Function (Like your multiply function!)

Just like you made multiply(x, y), let's make drawColorRectangle(x, color):

// Breaking it into a function (like you did with multiply!) function drawColorRectangle(x, color) { ctx.fillStyle = color; ctx.fillRect(x, 30, 80, 70); // x position changes, others stay same return color; // return the color (like multiply returned x*y) }
üîÑ Call Stack Thinking:
When we call drawColorRectangle(50, "red"):
1. Function gets called with x=50, color="red"
2. Sets fillStyle to "red"
3. Draws rectangle at position 50
4. Returns "red"
5. Function completes and pops off stack

Step 3: Draw Multiple Rectangles Using a Loop

Now let's call our function multiple times (like calling functions in your call stack example):

// Array of colors (like your a,b,c values in isRightAngle) const colors = ["red", "blue", "green", "orange", "purple"]; // Loop to draw each one (calling our function multiple times) function drawMultipleRectangles() { for (let i = 0; i < colors.length; i++) { drawColorRectangle(i * 80, colors[i]); // x position = i * 80 } }
üîÑ Call Stack for Loop:
Loop calls drawColorRectangle 5 times:
‚Ä¢ drawColorRectangle(0, "red") ‚Üí draws at x=0
‚Ä¢ drawColorRectangle(80, "blue") ‚Üí draws at x=80
‚Ä¢ drawColorRectangle(160, "green") ‚Üí draws at x=160
‚Ä¢ And so on... each function call completes before next one starts!

Step 4: Random Color Generator (New Function!)

Let's create a function that generates random colors (like your square function used multiply):

// Simple random color generator function generateRandomColor() { const colors = ["red", "blue", "green", "orange", "purple", "pink", "yellow", "cyan", "magenta", "lime"]; const randomIndex = Math.floor(Math.random() * colors.length); return colors[randomIndex]; // return random color } // Use it in our drawing function function drawRandomPalette() { for (let i = 0; i < 5; i++) { const randomColor = generateRandomColor(); // call our function! drawColorRectangle(i * 80, randomColor); } }
üéâ This is your basic palette generator! Each click gives you 5 random colors.
